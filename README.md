# Windows 8 Tab – Interactive Dashboard UI
## Project Overview

This project is a Windows 8-style interactive dashboard built using HTML, CSS, and Font Awesome. The dashboard features a tiled interface with hover animations and a responsive layout, providing a modern, visually engaging UI reminiscent of the Windows 8 start screen.

## Features

**Windows 8-style Tiles:** Interactive dashboard tiles representing apps and tools.
**Hover Animations:** Tiles enlarge smoothly on hover for a dynamic user experience.
**Responsive Layout:** Tiles and sections adapt to different screen sizes.
**Iconography:** Uses Font Awesome icons for apps like Camera, Browser, Facebook, and more.
**Event Tile:** Custom calendar tile displaying events and dates.
**Color-coded Tiles:** Distinct colors for different categories for better visual hierarchy.
**Footer:** Simple footer with author attribution.

## Technologies Used

- HTML5 – Semantic structure for tiles and layout.
- CSS3 – Styling, animations, and responsive design.
- Font Awesome – Icons for dashboard tiles.
- Segoe UI Font – Windows-inspired typography.

## Responsive Behavior

- Desktop: Tiles arranged in multiple rows with hover animations active.
- Tablet: Tiles adjust their width and spacing to fit smaller screens.
- Mobile: Tiles resize proportionally for readability, hover animations maintained where possible.

## How to Run

1. **Clone the repository:**
  git clone https://github.com/haniboutlat/Windows-8-Tab-Interactive-Dashboard-UI.git
2. Open index.html in your preferred web browser.
3. Interact with the dashboard and hover over tiles to see animations.
